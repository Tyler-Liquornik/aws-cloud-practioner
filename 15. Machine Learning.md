
AWS has a number of managed services for machine learning:
- **AWS Rekognition**
	- Find objects, people, text, scenes, in images and videos
	- Facial analysis and face search
	- Use cases: labeling, content moderation, text detection, face detection/analysis, pathing
- **AWS Transcribe**
	- Automatically convert speech to text
	- Uses a deep learning process called automatic speech recognition (ASR)
	- Can automatically remove PII
	- Supports automatic language identification for multi-lingual audio
	- Use cases: customer service calls, closed captioning, metadata for media assets
- **AWS Polly**
	- Turn text into speech using deep learning
	- Create apps that talk
- **AWS Translate**
	- Accurate language translation
	- Use to localize content for websites and apps for international users
- **AWS Lex**
	- Same speech to text tech that powers Amazon Alexa
	- Natural Language understanding of context / intent of text
	- For building chatbots or call center bots, when used with AWS Connect
- **AWS Connect**
	- Receive calls, create contact flows, in a virtual contact centre
	- Integrates with 3rd party CRMs or AWS
	- No upfront payments, 80% cheaper than traditional contact centre solutions
	- For building chatbots or call center bots, when used with AWS Lex
- **AWS Comprehend**
	- Fully managed and serverless NLP
	- Find insights and relationship in text
		- Language
		- Key phrases, places, people, brands, events
		- Sentiment analysis
		- Tokenization
	- Use cases: sentiment analysis on customer emails, group articles by topics
- **AWS SageMaker AI**
	- Fully managed service to build ML models
	- Provides managed training infrastructure (CPU/GPU)
    - Supports built-in algorithms and custom models
    - Handles automatic scaling for model endpoints
    - Includes tools for data labeling, experimentation, model tuning, and more
- **AWS Kendra**
	- Fully managed document semantic search engine service
	- Extract answers from within a document
	- Natural language search
	- Learns from user interactions/feedback to promote preferred results (incremental learning)
- **AWS Personalize**
	- Fully managed service to build apps with real-time personalization recommendations
	- Personalized product reommendations/re-ranking
	- Same tech used by amazon.com for e-commerce recommendations
	- Use cases: Retail stores, media and entertainment, ...
- **AWS Textract**
	- Automatically extract text, handwriting, and data from scanned documents
	- Can handle forms and tables
	- Can handle different document types (e.g., PDFs, images, etc.)
	- Use cases: financial services, healthcare, public sector
- **AWS Bedrock**
    - Fully managed service to build and scale generative AI applications
    - Provides access to foundation models (FMs) from AWS and third parties (e.g., Anthropic, Meta)
    - Supports text generation, summarization, chat, image generation, and RAG/embeddings
